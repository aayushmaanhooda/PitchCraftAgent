# PitchCraft

Turns an RFP into a polished sales response — paste the RFP text and get back an Excel compliance matrix and a tailored PowerPoint pitch deck.

The Excel agent drafts a questionnaire-style response grid; the PPT agent researches the prospect with Tavily, plans each slide's layout and visuals via an LLM, generates images (OpenAI / FLUX), and renders a branded `.pptx`. Built with FastAPI + React + Neon Postgres.
